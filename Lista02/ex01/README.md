ex01 — Conta bancária
📌 Objetivo

Praticar encapsulamento e regras de negócio.

📝 Enunciado

Crie uma classe ContaBancaria.

Ela deve possuir:

titular
saldo

Implemente os métodos:

depositar()
sacar()
exibirSaldo()
⚠️ Regras
Não permitir saque maior que saldo
Saldo deve ser privado
Não permitir depósito negativo
