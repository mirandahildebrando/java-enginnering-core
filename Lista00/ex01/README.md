# ex01 — Informações do usuário

## 📌 Descrição
Crie um programa que armazene e exiba as seguintes informações:

- Nome
- Idade
- Altura

## 📤 Saída esperada
Nome: X  
Idade: Y  
Altura: Z  

## ⚠️ Regras
- Todos os dados devem ser armazenados em variáveis
- A saída deve seguir exatamente o formato especificado
