# 🚀 Java Engineering Core

## 📌 Sobre o projeto

O **Java Engineering Core** é uma trilha completa de formação em desenvolvimento backend, inspirada na metodologia prática da 42, porém expandida para atender às exigências reais do mercado.

Diferente do comum, este projeto recria a experiência intensa da formação da 42 **em Java**, evoluindo do zero absoluto até o nível profissional.

A progressão foi pensada de forma intencional:

➡️ **Fundamentos e lógica primeiro (base sólida)**
➡️ **Depois Programação Orientada a Objetos**
➡️ **Depois frameworks e backend real**
➡️ **Por fim, deploy e desafios técnicos de entrevistas**

A proposta não é pular etapas, mas construir domínio real:

* Primeiro entender profundamente como resolver problemas
* Depois estruturar código
* Só então usar abstrações do mercado

Este repositório documenta toda a jornada, do básico ao avançado.

---

## 🧠 Filosofia da formação

Este projeto segue princípios claros:

* ✅ Base forte antes de abstração
* ❌ Não pular direto para frameworks
* ✅ Resolver problemas manualmente antes de otimizar
* ✅ Evolução progressiva e consistente
* ✅ Aprender fazendo (hands-on)
* ✅ Documentar evolução publicamente

Aqui, lógica não é o fim — é a fundação de tudo que vem depois.

## 🎯 Objetivo final

O objetivo desta trilha é atingir um nível profissional capaz de:

- Construir APIs REST completas com Java
- Trabalhar com Spring Boot e boas práticas
- Modelar sistemas reais
- Resolver desafios técnicos de entrevistas
- Fazer deploy de aplicações (backend + banco)
